# 🎮 Sora Escape

Sora Escape, Swift ve SwiftUI öğrenme sürecimde geliştirdiğim basit ama öğretici bir mobil oyundur.
Bu proje, iOS geliştirme dünyasına giriş yaparken temel kavramları öğrenmek ve uygulamak amacıyla yapılmıştır.

---

## 🎯 Projenin Amacı

Bu projeyi geliştirirken hedefim:

* Swift dilini öğrenmek
* SwiftUI ile arayüz geliştirmeyi kavramak
* Oyun mantığı (game loop, collision, state management) üzerine çalışmak
* Gerçek bir proje geliştirerek pratik kazanmak

---

## 🧠 Bu Projede Neler Öğrendim?

### 🔹 Swift & SwiftUI

* `@State` kullanımı ve state yönetimi
* View yapısı ve component mantığı
* Modifier’lar (`.font`, `.background`, `.overlay` vs.)

### 🔹 Oyun Mekanikleri

* Timer ile game loop oluşturma
* Dinamik obstacle (engel) üretimi
* Collision detection (çarpışma sistemi)
* Hitbox mantığı ve optimizasyonu

### 🔹 UI/UX

* Responsive layout (farklı ekranlara uyum)
* Overlay yapıları (pause, game over)
* Basit animasyonlar ve geçişler

### 🔹 Debug & Problem Çözme

* “Değmeden ölme” bug’ını hitbox küçülterek çözme
* Pause sisteminde state yönetimi
* UI katman sıralaması (ZStack mantığı)

---

## 🎮 Oyun Özellikleri
